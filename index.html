This is the Akan Website

This is a Website that helps the user to know the day of the week on which the person was born. The website is able to use the User's birthday to know the day. Then depending on the gender the user inputs ,their Akan names appears on clicking the submit button.

Akan names are came from a Ghanian culture. According to this Ghanian culture, after mothers gave birth to their children they named them their first names in accordance to the day they were born.
